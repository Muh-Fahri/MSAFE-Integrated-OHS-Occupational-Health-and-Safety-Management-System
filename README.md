MSAFE - Integrated OHS Management System
Professional Occupational Health and Safety Management System for Industrial Operations

📌 Project Overview
MSAFE adalah platform manajemen digital yang dirancang untuk mengotomatisasi seluruh proses Kesehatan dan Keselamatan Kerja (K3) di lingkungan industri. Sistem ini fokus pada efisiensi pelaporan insiden dan manajemen data personil secara terpusat.

Sebagai pengembang utama untuk Backend dan Frontend, saya membangun sistem ini untuk menggantikan proses manual menjadi alur kerja digital yang lebih cepat, transparan, dan terorganisir.

🚀 Key Features & Modules
Incident Reporting & Workflow: Sistem pelaporan insiden digital dengan alur investigasi dan tindak lanjut (Corrective Action) yang terdokumentasi dengan baik.

Personnel & Badge Management: Database terpusat untuk profil karyawan, penugasan kerja, serta sistem pengajuan identitas (Badge Request).

Automated OHS Notifications: Notifikasi sistem untuk setiap laporan insiden baru guna mempercepat respon tim di lapangan.

Compliance Documentation: Manajemen penyimpanan dokumen kepatuhan K3 dan log aktivitas harian yang aman.

Safety Analytics Dashboard: Visualisasi data statistik untuk memantau performa keselamatan kerja (Key Performance Indicators).

🛠 Tech Stack
Backend: Laravel Framework (PHP)

Frontend: Blade Templating, JavaScript, Bootstrap (Based on Minible Template)

Database: MySQL / MariaDB (Relational Database)

Server Environment: Laragon / Apache

👨‍💻 My Role & Contributions (Fullstack)
Fullstack Development: Mengembangkan seluruh logika sisi server (Backend) dan tampilan antarmuka pengguna (Frontend).

Database Engineering: Merancang skema database MySQL yang efisien untuk relasi data karyawan, departemen, dan log insiden.

Business Logic Implementation: Menerjemahkan prosedur operasional K3 ke dalam alur kerja sistem (Workflow) yang sistematis.

Access Control: Mengimplementasikan keamanan sistem berbasis peran (RBAC) untuk memastikan hak akses data yang tepat sesuai otoritas pengguna.
